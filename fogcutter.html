Fogcutter

Winner of the 2021 BOOST 5G Innovation Camp

In 2021 I entered in the BOOST 5G Innovation Camp hackathon where we were tasked with creating a pitch for a Mixed Reality application that used 5G networking to assist first responders. My group created and implemented the demo of an app named Fogcutter for the Hololens 2, developed in Unity using MRTK2. This was some of my first exposure to development within MRTK2, something which I have practiced since. The app itself was intended to be a Mixed Reality IoT application that created a realtime representation of a first responder situation and displayed data that would help clear confusion.
